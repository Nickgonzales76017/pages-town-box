Town in a Box

Meetings, hotlines, issues, and archives — a digital civic OS from one repo

How it works

Drop files in input/, commit, and watch Git hooks run the Bonfyre pipeline. Pages rebuild automatically.

Architecture

git commit → post-commit hook → Bonfyre binaries → site/ → GitHub Pages

Powered by Bonfyre — 47 C11 binaries, ~2.1 MB.
